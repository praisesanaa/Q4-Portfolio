What is Intro to Telecommunication Systems?

The skills that you will learn in this program are highly desirable in the telecommunications realm. The systems that provide broadband connectivity to homes and businesses and their applications are the focus of this program. This program provides the initial investigation into Telecommunications Technologies. These systems can be wired with copper, Fiber Optic cable or wireless. This program provides the foundational training.
